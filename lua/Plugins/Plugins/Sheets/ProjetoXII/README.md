 #### Comandos básicos
 * (Uso o IDE `VISUAL CODE`) mas pode usar prompt no diretório.
 * Segue os comandos
 ##### Atualiza modulo
  
  ```
  rdk p
  ```
  
   ##### Compila
  
  ```
  rdk c
  ```
   #####  Compila e instala
  
  ```
  rdk i
  ```
